## 背景

对应一般单页面应用（SPA）而言，一个路由对应一个页面，路由的切换会导致上一个页面被卸载（unmount），但是需要一种路由切换后，可以允许多个页面并存的机制，从而让应用更友好，生产效率更高。

## multiple page routing 定义（Definition）

multiple page routing（多页面并存的路由机制），指一个单页面应用中，由路由导航打开的页面并存的路由机制。
